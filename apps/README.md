# apps

This folder groups the runnable Spring Boot applications in the monorepo.

## Modules

- [flowable-engine](flowable-engine) - Flowable Engine sample backed by MySQL and Liquibase
- [gateway](gateway) - Reactive Spring Cloud Gateway sample
- [oauth2-authorization-server](oauth2-authorization-server) - JWT-focused Spring Authorization Server sample
- [oauth2-authorization-server-opaque](oauth2-authorization-server-opaque) - Opaque-token Spring Authorization Server sample
- [oauth2-gateway-mvc](oauth2-gateway-mvc) - Minimal Spring Cloud Gateway Server MVC sample
- [spring-security](spring-security) - Servlet resource server sample
- [spring-security-opaque](spring-security-opaque) - Servlet opaque-token resource server sample
- [spring-security-reactive](spring-security-reactive) - Reactive resource server sample
- [spring-security-reactive-opaque](spring-security-reactive-opaque) - Reactive opaque-token resource server sample
- [todo-app](todo-app) - Todo API sample

For shared local ports and supporting-service ports, see the [root port catalog](../README.md#local-profiles-and-ports).
