# templates

This folder groups reusable Spring Boot templates to build new applications. These templates are not meant to be used
directly, but rather to be copied and customized for each new application.

## Modules

- [oauth2-autoconfig](oauth2-autoconfig) - Shared servlet/reactive OAuth2 and security auto-configuration
- [test-controller-autoconfig](test-controller-autoconfig) - Small auto-configured test controller used by multiple apps
